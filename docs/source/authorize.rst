GET: Authorize
==============

HTTP METHOD - **GET**
Request URL - /v2/authorize/:userid

.. list-table:: **Form-Body**
   :widths: 25 25 50
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - userid
     - string
     - The userid that will be granted the api token. Verification will be sent to user with this id.
     
**RETURNS**

`200`_
